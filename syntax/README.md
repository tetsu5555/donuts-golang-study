# Go言語の基本文法

## 実行方法

```
$ go run [実行したいファイル名]
```

## 項目

1. `./var.go`: 変数宣言、代入
    - `./var_ex1.go`: 演習1
    - `./var_ex2.go`: 演習2
2. `./if.go`: 条件分・switch文
    - `./if_ex1.go`: 演習3
    - `./if_ex2.go`: 演習4
3. `./slice.go`: スライス
4. `./for.go`: 繰り返し文
    - `./for_ex1.go`: 演習5
5. `./func.go`: 関数
    - `./func_ex1.go`: 演習6
    - `./func_ex2.go`: 演習7
6. `./map.go`: 連想配列
    - `./map_ex1.go`: 演習8
